# freshcart

**a single-page web application created with ReactJS.**
Features:

- Authentication and Authorization
- get,delete,put and post data from Api
- crud operation on cart component

**Technologies:**

- React.js
- Context
- react-router-dom
- react-Query
- react hooks( useState-useContext-useEffect )
- useformik
- react-slick
- axios
- formik
- yup
- react-slick
- slick-carousel
- react-hot-toast
